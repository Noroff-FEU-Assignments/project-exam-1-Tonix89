Purpose

Daily Devotion is a blog website created for my semester project in Front-end development course. In this semester project, we need to put into practice the skills learned over our first year of studies. To be able to build a website using the fundamental programming language in building the website, HTML, CSS and JavaScript

Daily Devotion blog is inspired by Filadelifia Internation Church - Oslo corporate online prayer meeting. Where Christians all over the world meet digitally to encourage, share the Word of God, and pray for each other.
